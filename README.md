# MabiPaker v2

This application is a tool to import and export client data files (* .pack) of Nexon's online game "Mabinogi".

このアプリケーションはNexonのオンラインゲーム「Mabinogi」のクライアントのデータファイル（*.pack）を読み込んだり、書き出したりするためのツールです。
